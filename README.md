# Prime Video Movies & TV Shows Insights Dashboard using Excel - Project
This project presents an interactive Excel dashboard designed to analyze Prime Video content data from 1920 to 2021. The dashboard provides a comprehensive overview of total titles, movies, TV shows, directors, genres, and ratings available on the platform.
## key metrics
*This project presents an interactive Excel dashboard designed to analyze Prime Video content data from 1920 to 2021. The dashboard provides a comprehensive overview of total titles, movies, TV shows, directors, genres, and ratings available on the platform.

*The report highlights key performance indicators such as 9,668 total titles, 7,586 directors, and 519 genres. It visually represents the Top 10 Genres by total movies, helping identify the most dominant content categories like Drama and Comedy.

*A rating analysis section showcases the distribution of content across age categories such as 13+, 16+, and 18+, enabling better understanding of audience targeting. The country-wise map visualization provides insights into global content distribution.

*The release year trend analysis demonstrates the growth of content production over time, especially the sharp rise after 2000. A donut chart clearly differentiates between Movies and TV Shows for quick comparison.

*Advanced Excel features such as Pivot Tables, Pivot Charts, Slicers, Data Cleaning, and Conditional Formatting were used to build this dashboard. This project demonstrates strong data visualization, reporting, and analytical skills using Microsoft Excel.
